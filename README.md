## DM Editor monorepo sample project

Use npm workspace for monorepo. Frontend uses nextjs and backend use create-react-app

### Install
```sh
npm install

cd admin
npm install

cd site
npm install
```

### Run

Admin
```sh
cd admin
npm start
# visit http://localhost:3000
```

Frontend
```sh
cd site
npm run dev
# visit http://localhost:3000
```
